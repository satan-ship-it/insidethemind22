The Architecture of Memory

What is Memory?

This application provides an interactive exploration of human memory, based on established psychological concepts. Memory is the faculty that allows us to retain and recall information over time. It is central to nearly all cognitive processes, including perception, thinking, and problem-solving, and is fundamental to our sense of self.

Psychologists study how information is committed to memory, the mechanisms by which it is retained, the reasons for its loss, and techniques for improvement. This guide breaks down these complex topics into key areas you can explore.

The Three Foundational Processes

All information must pass through three interrelated stages to be successfully remembered. A failure at any stage can result in memory loss.

1. Encoding

The first stage where information is recorded and registered for the first time. External stimuli are received by sensory organs, converted into neural impulses, and represented in a way that the memory system can use.

2. Storage

The second stage. This is the process through which encoded information is retained and held over a period of time so that it can be put to use later.

3. Retrieval

The third stage. This refers to recovering the stored information and bringing it into awareness so it can be used for cognitive tasks like problem-solving or decision-making.

Forgetting & Memory Flaws

Forgetting is the loss of stored information. The first systematic exploration of memory by Hermann Ebbinghaus revealed that forgetting happens rapidly at first and then levels off. Beyond simple decay, memory can also be distorted or fail due to interference or flawed retrieval processes.

Ebbinghaus's Forgetting Curve

This chart illustrates the findings of Hermann Ebbinghaus, showing how quickly we forget new, meaningless information. The rate of forgetting is steepest in the first few hours and days.
